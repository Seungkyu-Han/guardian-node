# @seungkyu/guardian 🛡️

[![npm version](https://img.shields.io/npm/v/@seungkyu/guardian.svg)](https://www.npmjs.com/package/@seungkyu/guardian)
[![license](https://img.shields.io/npm/l/@seungkyu/guardian.svg)](https://github.com/Seungkyu-Han/guardian-node/blob/main/LICENSE)

A lightweight and robust JWT & Authentication utility for modern backend applications, optimized for NestJS ecosystems.

---

## ✨ Features

- 🔑 **Simple JWT Management**: Easily generate, verify, and decode JWT tokens.
- ⚡ **NestJS Integration**: Built on top of `@nestjs/jwt` for seamless integration into NestJS architecture.
- ⏱️ **Flexible Expiration**: Smart time string parsing support (powered by `ms`).
- 🛡️ **Type-Safe**: Full TypeScript support with comprehensive type definitions included out of the box.

---

## 📦 Installation

```bash
# npm
npm install @seungkyu/guardian

# yarn
yarn add @seungkyu/guardian

# pnpm
pnpm add @seungkyu/guardian

```
